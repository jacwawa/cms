# Fountain Snippets And Reference

Use this file as your operational reference for copy/paste standards.

## Image Format Rules

- Accepted file types: `JPG`, `PNG`

## Core Crop Sizes

- Show pages and old series pages: `1600x900`
- Calendar listings: `650x366`
- LincolnCenter.org page templates: `1673x600`
- New series header art: `1756x408`
- New sub-series header art: `1756x615`
- Series landing circular modules: `500x500`
- Series landing square modules: `600x600`
- Collections marketing module: `700x700`
- Press room: `906x453`

## Ticketing Copy Snippet

```html
<p>Ticketing information to be announced at a later date; please check back closer to the event.</p>
```

## No Late Seating Copy Snippet

```text
Please note: There will be no late seating; ticket-holders, please arrive on time.
```
