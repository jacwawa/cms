# Fountain Troubleshooting

## Service Unavailable Error

If only you are seeing it:

1. Clear browser cache.
2. Open `Create New Show` and test if it loads.
3. Log out and back in.

## Preview Not Matching Expected Template

1. Save the page.
2. Re-open and click Preview.
3. Replace URL path with expected template pattern (Atrium, Summer in the City, venue template).

## Broken PDF Link After Upload

Cause:

- Asset may have been re-saved and URL changed.

Fix:

1. Open asset record.
2. Copy current `File Location` URL.
3. Replace broken link in content.
4. Re-test link.

## Missing Show In Search

1. Enable `Show past shows` in filters.
2. Re-run search.

## Ticketing Button Appears Incorrectly

1. Verify single-date vs multi-date setup.
2. Single-date: button URL in Ticketing tab.
3. Multi-date: first URL in Basics, date-specific URLs in Dates tab.

## Image Not Correct On Mobile

1. Open image asset manager after initial save.
2. Upload alternate mobile image.
3. Re-test on mobile viewport.
