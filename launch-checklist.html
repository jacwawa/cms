Launch Checklist

Run this before switching visibility to live.

Overview Quickstart Workflows Troubleshooting Snippets Launch Checklist

Content

  • Show title and supporting H3/H3_2 are complete
  • Date line follows house style (:00, clean en dashes)
  • Description reviewed for accuracy
  • Button URL points to correct destination

Media

  • Desktop image is correctly cropped
  • Mobile image uploaded when needed
  • Caption and photo credit are present
  • PDFs are linked with tested URLs

Metadata / Controls

  • Display in Section is correct (Events for live events)
  • Is Featured set intentionally
  • Hide from On Demand set correctly
  • Add to Calendar enabled when appropriate
  • Tags (Free, Family, venue, series, genre) are complete

Ticketing

  • Single-date vs multi-date setup is correct
  • Ticket URLs tested
  • Ticket copy/language reviewed

QA and Approval

  • Page preview checked in correct template
  • Internal and external links tested
  • Stakeholder approval confirmed
  • Visibility switched from Invisible only after approval
Use this checklist in every launch handoff.
